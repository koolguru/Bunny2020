This is inspired by the bunnylol search engine at facebook. Bunny is a Shortcut Tool This iteration is primarily based on Jack Yang's jack_bunny which in turn was released by Charlie Cheever at bunny1. This iteration has a couple advantages over jack_bunny. Commands are now decoupled from the overall implementation and are now standalone classes. Each command can now implement much more complex functionality and can have multiple keywords. In this way it maps more closely to Facebook's current implementation. Adding commands is as simple as adding a new file in the Commands folder and overriding the Command class.

What can Bunny2020 do for me?

  • Enter g [query] to automatically carry out a Google search
  • Enter spotify [query] to automatically search spotify
  • Enter sound [query] to access the hottest drops from the latest artists
  • Anything else you may dream up!
  • Type 'help' into your address bar to see which commands are currently available

      • Installing on Google Chrome (Local run)

      • Clone the repo, run bash local_run.sh to start a local flask server
      • Open settings. Scroll down to the section where it says "Search engine"
      • Click 'Manage search engines'. Next to 'Other search engines', click the ADD button
      • Name the 'Search engine' and 'Keyword' field whatever you want. Inside the URL field, paste this: http://http://127.0.0.1:5000/q/?query=%s
      • Set this search engine to your default
